# MAC-TSC-ARGUETA

Repositorio que contiene múltiples proyectos de desarrollo.

## Proyectos

### TiendaUno
Proyecto de tienda online desarrollado con Spring Boot.

## Estructura
```
MAC-TSC-ARGUETA/
├── TiendaUno/          # Proyecto de tienda
└── [otros proyectos]   # Futuros proyectos se agregarán aquí
```

## Cómo agregar nuevos proyectos

Simplemente crea una nueva carpeta en la raíz del repositorio para cada proyecto nuevo:
```bash
mkdir NuevoProyecto
cd NuevoProyecto
# Inicializa tu proyecto aquí
```
